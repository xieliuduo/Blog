                    技巧 1： 支持RESTful URL，如果在URL填写:key，然后在Query参数中填写对应 key，发送请求的时候插件将会替换URL中的 :key 
        
                    
                
        
                
        
                    技巧 2： 支持全局环境变量，如果在URL或Query或Body或Header 里填写{{key}}，发送请求的时候插件将会用环境变量中key的值替换{{key}}
